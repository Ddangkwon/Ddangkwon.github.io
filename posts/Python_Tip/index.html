  1. 리스트 언패킹으로 괄호 제거 ([1, 2, 3] [4, 5, 6] [7, 8, 9])
  2. zip([], [], []) : zip 함수 내에 3개씩의 정수를 가진 세 개의 리스트가 들어가있는 형태
  3. 각각의 리스트에서 한개씩 꺼내 3개의 튜플을 반환 ([(1, 4, 7), (2, 5, 8), (3, 6, 9)])
  4. map 함수로 리스트화 [[1, 4, 7], [2, 5, 8], [3, 6, 9]]

zip 함수 세부 활용법

zip 함수 설명 => zip(*iterables)는 각 iterables 의 요소들을 모으는 이터레이터를 만듭니다. 튜플의 이터레이터를 돌려주는데, i 번째 튜플은 각 인자로 전달된 시퀀스나 이터러블의 i 번째 요소를 포함합니다.

    answer = []
