상어 초등학교

1, 비어있는 칸 중에서 좋아하는 학생이 인접한 칸에 가장 많은 칸으로 자리를 정한다.

  1. 1을 만족하는 칸이 여러 개이면, 인접한 칸 중에서 비어있는 칸이 가장 많은 칸으로 자리를 정한다.
  2. 2를 만족하는 칸도 여러 개인 경우에는 행의 번호가 가장 작은 칸으로, 그러한 칸도 여러 개이면 열의 번호가 가장 작은 칸으로 자리를 정한다.

학생 입력 순서대로 빈 공간 중 가장 선호도가 높은 자리에 배치하고, 모든 배치가 끝나면 선호도 점수를 계산한다.

// 최종적인 만족도를 계산하는 함수
int calc_point(int idx)
{
	int point = 0;
	for(int i = 0; i < N; i++)
	{
		for(int j = 0; j < N; j++)
		{
			int temp_point = 1;
			int cnt = 0;
			int cy = stud[idx].y;
			int cx = stud[idx].x;
			for(int dir = 0; dir < 4; dir++)
			{
				int ny = cy + dy[dir];
				int nx = cx + dx[dir];
				for(int k = 0; k < 4; k++)
				{
					if(arr[ny][nx] == stud[idx].friend[k])
					{
						cnt++;
						temp_point *= 10;
					}
				}
			}
		}
		if(cnt == 0)
		{
			temp_point = 0;
		}
		point +=  temp_point;
	}

	return point;
}
