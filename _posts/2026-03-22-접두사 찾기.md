# 전화번호목록

## 문제
문자열 S의 접두사란 S의 가장 앞에서부터 부분 문자열을 의미한다. 예를 들어, S = "codeplus"의 접두사는 "code", "co", "codepl", "codeplus"가 있고, "plus", "s", "cude", "crud"는 접두사가 아니다.

총 N개의 문자열로 이루어진 집합 S가 주어진다.

입력으로 주어지는 M개의 문자열 중에서 집합 S에 포함되어 있는 문자열 중 적어도 하나의 접두사인 것의 개수를 구하는 프로그램을 작성하시오.

## 입력
첫째 줄에 문자열의 개수 N과 M (1 ≤ N ≤ 10,000, 1 ≤ M ≤ 10,000)이 주어진다.

다음 N개의 줄에는 집합 S에 포함되어 있는 문자열이 주어진다.

다음 M개의 줄에는 검사해야 하는 문자열이 주어진다.

입력으로 주어지는 문자열은 알파벳 소문자로만 이루어져 있으며, 길이는 500을 넘지 않는다. 집합 S에 같은 문자열이 여러 번 주어지는 경우는 없다.

## 출력
첫째 줄에 M개의 문자열 중에 총 몇 개가 포함되어 있는 문자열 중 적어도 하나의 접두사인지 출력한다.

## 풀이

```cpp
#include <iostream>
#include <vector>
#include <string>

using namespace std;
typedef struct{
    int path_cnt;
    int child[26];
}trie_node_t;

vector<trie_node_t> buf;
int buf_cnt = 0;
int N, M, answer = 0;


void push(string str)
{
    int cur_buf_cnt = 0;

    for(int i = 0; i < str.length(); i++){
        int idx = str[i] - 'a';
        // 해당 접두사가 처음 등장한 경우
        if(buf[cur_buf_cnt].child[idx] == 0){
            // 새로운 노드 생성 후 번호 지정
            buf.push_back({ 0 });
            buf[cur_buf_cnt].child[idx] = ++buf_cnt;
        }
        //cur_buf_cnt, path_cnt 업데이트
        cur_buf_cnt = buf[cur_buf_cnt].child[idx];
        buf[cur_buf_cnt].path_cnt++;
    }
}

int get_path_cnt(string str)
{
    int ret_buf_cnt = 0;

    for(int i = 0; i < str.length(); i++){
        int idx = str[i] - 'a';
        // break 로 할 경우 반례 -> codx (마지막 끝 글자만 틀릴 경우 break 해서 cod 의 path_cnt 값을 리턴해버림)
        if(buf[ret_buf_cnt].child[idx] == 0)
            return 0;
        // ret_buf_cnt 갱신
        ret_buf_cnt = buf[ret_buf_cnt].child[idx];
    }

    return buf[ret_buf_cnt].path_cnt;
}

int main(void)
{
    ios::sync_with_stdio(false);
    cin.tie(NULL);
    cout.tie(NULL);

    string in_str;
    cin >> N >> M;

    // init
    buf.push_back({ 0 });
    for(int i = 0; i < N; i++){
        cin >> in_str;
        push(in_str);
    }

    for(int i = 0; i < M; i++){
        int ret = 0;
        cin >> in_str;
        ret = get_path_cnt(in_str);
        if(ret)
            answer++;
    }

    cout << answer << "\n";
    return 0;
}
```
