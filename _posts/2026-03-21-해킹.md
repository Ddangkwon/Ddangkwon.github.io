# 해킹

## 문제
최흉최악의 해커 yum3이 네트워크 시설의 한 컴퓨터를 해킹했다! 이제 서로에 의존하는 컴퓨터들은 점차 하나둘 전염되기 시작한다. 어떤 컴퓨터 a가 다른 컴퓨터 b에 의존한다면, b가 감염되면 그로부터 일정 시간 뒤 a도 감염되고 만다. 이때 b가 a를 의존하지 않는다면, a가 감염되더라도 b는 안전하다.

최흉최악의 해커 yum3이 해킹한 컴퓨터 번호와 각 의존성이 주어질 때, 해킹당한 컴퓨터까지 포함하여 총 몇 대의 컴퓨터가 감염되며 그에 걸리는 시간이 얼마인지 구하는 프로그램을 작성하시오.

## 입력
첫째 줄에 테스트 케이스의 개수가 주어진다. 테스트 케이스의 개수는 최대 100개이다. 각 테스트 케이스는 다음과 같이 이루어져 있다.

첫째 줄에 컴퓨터 개수 n, 의존성 개수 d, 해킹당한 컴퓨터의 번호 c가 주어진다(1 ≤ n ≤ 10,000, 1 ≤ d ≤ 100,000, 1 ≤ c ≤ n).
이어서 d개의 줄에 각 의존성을 나타내는 정수 a, b, s가 주어진다(1 ≤ a, b ≤ n, a ≠ b, 0 ≤ s ≤ 1,000). 이는 컴퓨터 a가 컴퓨터 b를 의존하며, 컴퓨터 b가 감염되면 s초 후 컴퓨터 a도 감염됨을 뜻한다.
각 테스트 케이스에서 같은 의존성 (a, b)가 두 번 이상 존재하지 않는다.

## 출력
각 테스트 케이스마다 한 줄에 걸쳐 총 감염되는 컴퓨터 수, 마지막 컴퓨터가 감염되기까지 걸리는 시간을 공백으로 구분지어 출력한다..

## 풀이

```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <climits>

#define MAX_COMP_NUM 10005
using namespace std;
int tc;
int n, d, c;
typedef struct{
    int dest;
    int time;
}data_t;
vector<data_t> graph[MAX_COMP_NUM];
int dist[MAX_COMP_NUM];
int done[MAX_COMP_NUM];
struct compare{
    bool operator()(const data_t& a, const data_t& b) const{
        return a.time > b.time;
    }
};
int def_comp_cnt = 0;
int last_comp_def_time = 0;

void dijkstra(int start)
{
    priority_queue<data_t, vector<data_t>, compare> pq;
    pq.push({start, 0});
    dist[start] = 0;

    while(!pq.empty()){
        data_t cur_data = pq.top();
        int cur_dest = cur_data.dest;
        int cur_time = cur_data.time;

        pq.pop();

        if(dist[cur_dest] != cur_time)
            continue;
        if(done[cur_dest])
            continue;
        done[cur_dest] = true;
        def_comp_cnt++;
        last_comp_def_time = max(last_comp_def_time, dist[cur_dest]);
        for(auto& next_data : graph[cur_dest]){
            int next_dest = next_data.dest;
            int next_time = dist[cur_dest] + next_data.time;

            if(next_time < dist[next_dest]){
                pq.push({next_dest, next_time});
                dist[next_dest] = next_time;
            }
        }
    }
}
void simulate()
{

    cin >> n >> d >> c;
    // init seq
    for(int i = 0; i <= n; i++){
        graph[i].clear();
        dist[i] = INT_MAX;
        done[i] = false;
    }

    def_comp_cnt = 0;
    last_comp_def_time = 0;

    int a, b, s;
    for(int i = 0; i < d; i++){
        cin >> a >> b >> s;
        graph[b].push_back({a, s});
    }

    dijkstra(c);
}
int main(void)
{
    ios::sync_with_stdio(false);
    cin.tie(NULL);
    cout.tie(NULL);
    cin >> tc;
    for(int i = 0; i < tc; i++){
        simulate();

        cout << def_comp_cnt << " " << last_comp_def_time << "\n";
    }
    return 0;
}

```
