# N번째 큰 수 
우선순위 큐 문제로 우선 순위 큐에 데이터를 오름차순 정렬 형태로 채워 놓고,
큐의 사이즈가 N보다 커지면 하나씩 꺼내 최종적으로 큐에 N개만 남게한다.
이렇게 되면 Queue의 Top은 N 번째로 큰 숫자가 된다.

```cpp
#include <iostream>
#include <queue>
#include <vector>
#include <algorithm>

using namespace std;

int main(void)
{
	int N, x;
	priority_queue<int, vector<int>, greater<int>> pq;
	scanf("%d", &N);
	for (int i = 0; i < N * N; i++)
	{
		
		scanf("%d", &x);
		pq.push(x);
		if (pq.size() > N)
			pq.pop();
	}	
	cout << pq.top() << '\n';
}
```
