# 정수 사각형 최대 합

[https://www.codetree.ai/ko/trails/complete/curated-cards/intro-maximum-sum-path-in-square/description](https://www.codetree.ai/ko/trails/complete/curated-cards/intro-maximum-sum-path-in-square/description)

(1, 1) 에서 (N, N) 까지 이동하며 오른쪽, 밑으로만 이동할 때 거쳐간 값의 합의 최대값을 구하는 문제

## 풀이

```cpp
#include <iostream>

using namespace std;

int n, answer = 0;
int grid[100][100];
int dp[100][100] = { 0 };

void init(){
    dp[0][0] = grid[0][0];

    for(int i = 1; i < n; i++){
        dp[i][0] = dp[i - 1][0] + grid[i][0];
    }

    for(int i = 1; i < n; i++){
        dp[0][i] = dp[0][i - 1] + grid[0][i];
    }

}
int main() {

    ios::sync_with_stdio(false);
    cin.tie(NULL);
    cout.tie(NULL);

    cin >> n;

    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            cin >> grid[i][j];
        }
    }

    init();


    for (int i = 1; i < n; i++) {
        for (int j = 1; j < n; j++) {
            dp[i][j] = max((dp[i - 1][j] + grid[i][j]), (dp[i][j - 1] + grid[i][j]));
        }
    }


    cout << dp[n - 1][n - 1] << "\n";

    return 0;
}

```
