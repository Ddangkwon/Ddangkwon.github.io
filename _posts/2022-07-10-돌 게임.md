# 돌 게임
결과적으로 짝수와 홀수를 판단하는 문제로
무조건 1개 혹은 3개를 가져갈 수 있어 짝수일 때는 창영이, 홀수일 때는 상근이 이기게 된다.

```cpp
#include <iostream>
#include <cstdio>

using namespace std;

int main(void)
{
	int N;
	scanf("%d", &N);

	string answer = ((N & 1) == 0) ? "CY" : "SK";

	cout << answer << '\n';
	return 0;
}

```
