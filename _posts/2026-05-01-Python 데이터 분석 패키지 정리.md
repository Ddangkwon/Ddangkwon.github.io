# Python 데이터 분석 핵심 패키지 메서드 예시 정리

## 개요

Python 기반 데이터 분석에서 가장 자주 사용하는 핵심 패키지는 다음 네 가지이다.

| 패키지 | 주요 역할 | 대표 사용 목적 |
|---|---|---|
| pandas | 표 형태 데이터 처리 | CSV 읽기, 결측치 처리, 그룹 집계, 병합 |
| NumPy | 수치 연산 | 배열 연산, 벡터화, 통계 계산, 난수 생성 |
| SciPy | 과학 계산 및 통계 분석 | 통계 검정, 최적화, 보간, 거리 계산 |
| Matplotlib | 데이터 시각화 | 선 그래프, 산점도, 막대 그래프, 히스토그램 |

이 문서는 각 패키지의 개념을 길게 설명하기보다, 실제 분석에서 자주 쓰는 메서드를 예시 코드 중심으로 정리한다.

# 1. pandas

## 1.1 pandas 개념

pandas는 행과 열로 구성된 표 형태 데이터를 다루는 패키지이다.
Excel, CSV, SQL 테이블처럼 생긴 데이터를 Python에서 처리할 때 가장 많이 사용한다.

pandas의 핵심 자료구조는 두 가지이다.

| 자료구조 | 설명 |
|---|---|
| Series | 1차원 데이터 |
| DataFrame | 2차원 표 형태 데이터 |

## 1.3 DataFrame 생성

### 리스트와 딕셔너리로 DataFrame 만들기

```python
import pandas as pd

data = {
    "name": ["Kim", "Lee", "Park", "Choi"],
    "age": [25, 31, 29, 35],
    "score": [85, 92, 78, 88]
}

df = pd.DataFrame(data)

print(df)
```

출력 예시는 다음과 같다.


## 1.4 CSV 파일 읽기

### read_csv

CSV 파일을 DataFrame으로 읽을 때 사용한다.

```python

```

### 자주 사용하는 옵션

```python
import pandas as pd

df = pd.read_csv(
    "data.csv",
    encoding="utf-8",
    sep=",",
    header=0
)

print(df.head())
```

| 옵션 | 의미 |
|---|---|
| encoding | 파일 인코딩 방식 |
| sep | 컬럼 구분자 |
| header | 컬럼명으로 사용할 행 번호 |
| index_col | 인덱스로 사용할 컬럼 |
| usecols | 읽어올 컬럼 지정 |

## 1.25 apply 사용

## 1.26 groupby 집계


### 여러 컬럼 집계

```python

    "time": [10, 12, 9, 11, 8]
}

result = df.groupby("team").agg({
    "score": "mean",
    "time": "sum"
})

```

## 1.27 데이터 병합

### merge

SQL의 join과 비슷한 기능이다.

```python
left = pd.DataFrame({

    "name": ["Kim", "Lee", "Park"]
})

    "id": [1, 2, 4],
    "score": [90, 85, 70]
})

```

### left join

```python
merged = pd.merge(left, right, on="id", how="left")

print(merged)
```

### outer join

```python
merged = pd.merge(left, right, on="id", how="outer")

print(merged)
```

## 3.3 정규성 검정

### shapiro

데이터가 정규분포를 따른다고 볼 수 있는지 확인할 때 사용한다.

```python
from scipy import stats

data = [10, 12, 11, 13, 12, 14, 11, 10, 13, 12]

statistic, p_value = stats.shapiro(data)

print("statistic:", statistic)
print("p-value:", p_value)
```

일반적인 해석은 다음과 같다.

| p-value | 해석 |
|---|---|
| 0.05 이상 | 정규성을 크게 위반한다고 보기 어려움 |
| 0.05 미만 | 정규성을 만족하지 않을 가능성이 큼 |

## 3.5 독립 표본 t 검정

### ttest_ind

# 5. 간단한 데이터 분석 흐름 예시

## 5.1 분석 시나리오

학생들의 공부 시간과 시험 점수 데이터가 있다고 가정한다.
pandas로 데이터를 정리하고, NumPy로 기초 통계를 계산하고, SciPy로 상관분석과 회귀분석을 수행하고, Matplotlib으로 시각화한다.

---

## 5.2 전체 예시 코드

```python
import pandas as pd
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

data = {
    "student": ["A", "B", "C", "D", "E", "F"],
    "study_hours": [2, 3, 5, 6, 8, 9],
    "score": [60, 65, 75, 80, 88, 92]
}

df = pd.DataFrame(data)

print("DataFrame")
print(df)

print("Basic Statistics")
print(df[["study_hours", "score"]].describe())

mean_score = np.mean(df["score"])
std_score = np.std(df["score"])

print("Mean Score:", mean_score)
print("Standard Deviation:", std_score)

correlation, p_value = stats.pearsonr(df["study_hours"], df["score"])

print("Correlation:", correlation)
print("p-value:", p_value)

result = stats.linregress(df["study_hours"], df["score"])

print("Slope:", result.slope)
print("Intercept:", result.intercept)

predicted_score = result.slope * df["study_hours"] + result.intercept

plt.scatter(df["study_hours"], df["score"], label="Actual")
plt.plot(df["study_hours"], predicted_score, label="Regression Line")
plt.title("Study Hours and Score")
plt.xlabel("Study Hours")
plt.ylabel("Score")
plt.legend()
plt.grid(True)
plt.show()
```

# 8. 마무리

pandas, NumPy, SciPy, Matplotlib은 Python 데이터 분석의 기본 축이다.

pandas는 데이터를 표 형태로 정리하고, NumPy는 빠른 수치 연산을 담당한다.
SciPy는 통계 검정, 최적화, 과학 계산에 사용되며, Matplotlib은 분석 결과를 시각적으로 확인하는 데 사용된다.

실제 분석에서는 네 패키지를 따로 쓰기보다 함께 사용한다.
예를 들어 pandas로 CSV를 읽고, NumPy로 평균과 표준편차를 계산하고, SciPy로 통계 검정을 수행한 뒤, Matplotlib으로 결과를 시각화하는 흐름이 가장 일반적이다.
