# 게임을 만든 동준이
오름차순을 만드는 문제로 볼 수 있고 따라서, 문제의 접근을 뒤에서부터 가지면 좋다.
오름차순 =>앞의 숫자보다 뒤의 숫자가 무조건 크면 된다.
```cpp
#include <iostream>
#include <cstdio>
#include <algorithm>

int N;
int point[100] = { 0, };

int main(void)
{
	int answer = 0;
	scanf("%d", &N);
	for (int i = 0; i < N; i++)
	{
		scanf("%d", &point[i]);
	}

	for (int i = N - 1; i > 0; i--)
	{
		if (N == 1)
			break;

		while (point[i - 1] >= point[i])
		{
			point[i - 1]--;
			answer++;

		}
	}

	cout << answer << '\n';
	return 0;
}

```
