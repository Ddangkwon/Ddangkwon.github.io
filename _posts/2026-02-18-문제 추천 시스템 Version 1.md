# 문제 추천 시스템 Version 1

## 입력

1. 첫 줄에 초기 문제의 개수 **N** 이 주어진다.
2. 다음 **N** 줄에 각 문제의 정보가 주어진다.
   - 문제 번호 **P**, 난이도 **L**
3. 다음 줄에 명령의 개수 **M** 이 주어진다.
4. 다음 **M** 줄에 명령이 한 줄씩 주어진다.

### 명령 형식

- **recommend x**
  - x가 **1** 이면 가장 어려운 문제를 추천한다.
  - x가 **-1** 이면 가장 쉬운 문제를 추천한다.
  - 추천 기준
    - 가장 어려운 문제: 난이도 **최대**, 난이도 같으면 문제 번호 **최대**
    - 가장 쉬운 문제: 난이도 **최소**, 난이도 같으면 문제 번호 **최소**

- **add P L**
  - 문제 번호가 **P**, 난이도가 **L** 인 문제를 목록에 추가한다.

- **solved P**
  - 문제 번호가 **P** 인 문제를 목록에서 제거한다.

## 출력

- **recommend** 명령이 나올 때마다, 추천되는 **문제 번호 P** 를 한 줄에 하나씩 출력한다.


## 풀이

```cpp
#include <iostream>
#include <set>
#include <map>

#define MAX_N 100005
using namespace std;
using pii = pair<int, int>;

int N, L;

int main(void)
{
    ios::sync_with_stdio(false);
    cin.tie(NULL);
    cout.tie(NULL);
    set<pii> s; // level, problem_no
    map<int, int> m; // problem_no, level

    cin >> N;

    for(int i = 0; i < N; i++){
        int level, p_no;
        cin >> p_no >> level;
        s.insert({level, p_no});
        m[p_no] = level;
    }

    cin >> L;

    while(L--){
        string str;
        cin >> str;

        if(str == "recommend"){
            int x;
            cin >> x;

            if(x == 1){
                auto it = s.rbegin();
                cout << it->second << "\n";
            }
            else{
                auto it = s.begin();
                cout << it->second << "\n";
            }
        }
        else if(str == "add"){
            int p_no, level;
            cin >> p_no >> level;
            s.insert({level, p_no});
            m[p_no] = level;

        }
        else if(str == "solved"){
            int p_no;
            cin >> p_no;
            s.erase({m[p_no], p_no});
            m.erase(p_no);
        }
    }
    return 0;
}

```
