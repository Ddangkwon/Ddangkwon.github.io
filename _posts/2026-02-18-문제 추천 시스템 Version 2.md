# 문제 추천 시스템 Version 2

## 문제 요약

추천 문제 리스트에 (문제 번호 P, 난이도 L, 알고리즘 분류 G) 형태의 데이터가 들어 있다.
아래 명령들을 처리하면서, 추천 명령이 나올 때마다 조건에 맞는 문제 번호를 출력한다.

## 명령어 정리

### 1. recommend G x

- 알고리즘 분류가 G인 문제들 중에서 추천한다.
- x = 1 이면: 가장 어려운 문제 번호 출력
  - 난이도 L이 가장 큰 문제
  - 난이도가 같으면 문제 번호 P가 큰 것
- x = -1 이면: 가장 쉬운 문제 번호 출력
  - 난이도 L이 가장 작은 문제
  - 난이도가 같으면 문제 번호 P가 작은 것
- 이 명령은 항상 분류 G에 해당하는 문제가 1개 이상 존재하는 상황에서만 주어진다.

### 2. recommend2 x

- 알고리즘 분류와 무관하게(전체 문제 대상) 추천한다.
- x = 1 이면: 가장 어려운 문제 번호 출력
  - 난이도 L이 가장 큰 문제
  - 난이도가 같으면 문제 번호 P가 큰 것
- x = -1 이면: 가장 쉬운 문제 번호 출력
  - 난이도 L이 가장 작은 문제
  - 난이도가 같으면 문제 번호 P가 작은 것

### 3. recommend3 x L

- 알고리즘 분류와 무관하게(전체 문제 대상) 난이도 기준으로 추천한다.
- x = 1 이면: 난이도 L 이상인 문제들 중에서 가장 쉬운 문제 번호 출력
  - 난이도 L이 가장 작은 문제
  - 난이도가 같으면 문제 번호 P가 작은 것
  - 조건을 만족하는 문제가 없으면 -1 출력
- x = -1 이면: 난이도 L 미만인 문제들 중에서 가장 어려운 문제 번호 출력
  - 난이도 L이 가장 큰 문제
  - 난이도가 같으면 문제 번호 P가 큰 것
  - 조건을 만족하는 문제가 없으면 -1 출력

### 4. add P L G

- 추천 문제 리스트에 (P, L, G)를 추가한다.
- 입력으로 들어오는 P는 현재 추천 문제 리스트에 없는 값만 주어진다.
- 과거에 있었다가 삭제된 P가 다시 들어오는 경우는 있을 수 있다.

### 5. solved P

- 추천 문제 리스트에서 문제 번호 P를 제거한다.
- 이 명령으로 주어지는 P는 항상 추천 문제 리스트에 존재한다.

## 입력

첫 줄에 추천 문제 리스트에 들어 있는 문제 수 N이 주어진다.

다음 N줄에 걸쳐 각 문제의 정보가 주어진다.

    P L G

그 다음 줄에 명령의 개수 M이 주어진다.

다음 M줄에 걸쳐 명령이 주어진다.

## 출력

recommend, recommend2, recommend3 명령이 나올 때마다 조건에 맞는 문제 번호를 한 줄에 하나씩 출력한다.

## 제한

- 1 ≤ N ≤ 100,000
- 1 ≤ M ≤ 10,000
- 1 ≤ P ≤ 100,000
- 1 ≤ L, G ≤ 100 (자연수)
- x는 -1 또는 1
- recommend, recommend2, recommend3 명령은 전체 입력에서 최소 1번 이상 등장한다.



## 풀이

```cpp
#include <iostream>
#include <set>
#include <unordered_map>

#define MAX_N 100005

using namespace std;
using pii = pair<int, int>;

int N, L;
set<pii> s; // 난이도 문제번호
set<pii> sg[101]; // 그룹 별 저장 난이도 문제 번호, recommend G 를 처리하기 위해 사용, 이 경우 난이도 문제는 같더라도 그룹이 다른 경우를 구분할 수 있음
unordered_map<int, pii> um; //문제 번호, {난이도, 그룹}
int main(void)
{
    ios::sync_with_stdio(false);
    cin.tie(NULL);
    cout.tie(NULL);

    cin >> N;

    for(int i = 0; i < N; i++){
        int p, l, g;

        cin >> p >> l >> g;
        s.insert({l, p});
        sg[g].insert({l, p});
        um[p] = {l, g};
    }

    cin >> L;

    while(L--){
        string str;
        cin >> str;

        if(str == "recommend"){
            int g, x;
            cin >> g >> x;
            if(x == 1){
                auto it = sg[g].rbegin();
                cout << it->second << "\n";
            }
            else{
                auto it = sg[g].begin();
                cout << it->second << "\n";
            }
        }
        else if(str == "recommend2"){
            int x;
            cin >> x;
            if(x == 1){
                auto it = s.rbegin();
                cout << it->second << "\n";
            }
            else{
                auto it = s.begin();
                cout << it->second << "\n";
            }
        }
        else if(str == "recommend3"){
            int x, l;
            cin >> x >> l;
            // 난이도 L 이상 중: L 최소, 동률이면 P 최소
            if(x == 1){
                auto it = s.lower_bound({l, 0});
                if(it == s.end())
                    cout << "-1\n";
                else
                    cout << it->second << "\n";
            }
            // 난이도 L 미만 중: L 최대, 동률이면 P 최대
            else{
                auto it = s.lower_bound({l, 0});
                // L 미만인게 없을 때
                if(it == s.begin())
                    cout << "-1\n";
                else{
                    if(it == s.end()){
                        // 모두 L 미만인 경우 set 의 마지막 값이 정답
                        auto rit = s.rbegin();
                        cout << rit->second << "\n";
                    }
                    // L 이상이 하나라도 있는 경우 직전 원소가 L 미만 중 최대 값
                    else{
                        it--;
                        cout << it->second << "\n";
                    }
                }
            }
        }
        else if(str == "add"){
            int p, l, g;
            cin >> p >> l >> g;
            s.insert({l, p});
            sg[g].insert({l, p});
            um[p] = {l, g};
        }

        else if(str == "solved"){
            int p;
            cin >> p;
            auto it = um.find(p);
            if(it == um.end())
                continue;
            int l = it->second.first;
            int g = it->second.second;
            s.erase({l, p});
            sg[g].erase({l, p});
            um.erase(it);
        }
    }


    return 0;
}

```
